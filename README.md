# homebrew-lazyclaude

Homebrew tap for [lazyclaude](https://github.com/idossha/lazyclaude) — a lazygit-inspired TUI for managing Claude Code configuration.

## Install

```sh
brew install idossha/lazyclaude/lazyclaude
```

Or add the tap first:

```sh
brew tap idossha/lazyclaude
brew install lazyclaude
```

## Update

```sh
brew upgrade lazyclaude
```
